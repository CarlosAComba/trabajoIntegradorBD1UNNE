Tema: Índices columnares en SQL server

Objetivos:

Entender conceptualmente el significado.
Tareas: 

Crear un documento de acuerdo al modelo existente en el aula virtual.
Crear una nueva tabla, tomando como modelo la tabla gasto de nombre GASTONEW.
Realizar una carga masiva de por lo menos 1 millón de registro sobre la tabla recién creada. Se pueden repetir los registros ya existentes. Hacerlo con un script para poder compartirlo.
Definir un índice columnar sobre la tabla GASTONEW. 
Ejecutar una consulta sobre tabla gasto y la tabla GASTONEW, evaluar los tiempos de respuestas entre ambas tablas (Con índice columnar y sin el mismo).
Expresar las conclusiones en base a las pruebas realizadas.
